  • Variables
    JavaScript has dynamic types.
    var x = 5.
    JavaScript has only one type of numbers.
    By re-declaring a variable, it does not lose its value.
  • types
    There are 6 types: 4 primitive types (string, number, boolean, undefined), 2 complex types (function, object).
    A primitive data doesn't have any properties or methods.
    null and undefined both have value undefined, but null is of type object and undefined is of type undefined.
    Arrays and dictionaries (called 'objects' in JS) are of type object.
  • Objects
    Objects are key:value pairs (like dictionaries in Python).
    We can access object properties in two ways: objName.propertyName or objName["propertyName"]
    var person = {
    firstName: "John",
    lastName : "Doe",
    id       : 5566,
    fullName : function() {
        return this.firstName + " " + this.lastName;
};

    We can access object methods as objName.methodName(). (Leaving "()"" off will return the function definition).
  • Functions
    function myFunction(arg1, arg2) {
    ... ;
    return ...;
}

    If you assign a value to a variable that has not been declared, it will automatically become a GLOBAL variable (even in a function), unless JavaScript is running in "strict mode".
  • Events
    JS can react to HTML events such as a button is clicked, or a web page has finished loading.
    <button onclick="document.getElementById('demo').innerHTML=Date()">The time is?</button>

<p id="demo"></p>

    To sho the date inside the button: <button onclick="this.innerHTML=Date()">The time is?</button> Common events: onclick, onmouseover, onmouseout, onkeydown, onload, onchange ( full list).
  • Misc.
    - When adding a number and a string, JavaScript will treat the number as a string.
    - ===  equal value and equal type
    - ?  ternary operator
    - In HTML, the global scope is the window object. All global variables belong to the window object. Your global variables (or functions) can overwrite window variables (or functions). Any function, including the window object, can overwrite your global variables and functions.
